первая цифра означает к чему относиться код (всм подразделения пример: сообщения,регистрация)
список кодов

регистрация

901 = акаунт с таким логином уже существует
902 = для регистрации необходимо как минимум 4 пункта name,lastname,login,password

авторизация 
800 = авторизация успешна 
801 = неверный пароль
802 = учетной записи с таким логином не существует

токен
701 = токен не актуален
702 = токен равен null

Новости

603 = новостная лента не удалось получить
